About me

Olá mundo, me chamo Isaac, residente de Recife, estudante de análise e desenvolvimento de sistemas e buscando entrar no mercado de desenvolvimento de software. Esta página tem o objetivo de mostrar um pouco do meu conhecimento em algumas linguagens de programação e frameworks.

Realizei boa parte dos meus estudos em desenvolvimento web, tenho conhecimentos em lógica de programação, Python + testes unitários, Django Framework e JavaScript. Em um projeto de estudos desenvolvi um blog utiizando Django Framework, Bootstrap, CSS e AWS para servir meus arquivos de media, neste projeto optei por realizar o deploy no Heroku e assim aprimorar meus conhecimentos na plataforma. Estudei um pouco sobre REST API utilizando o Django Rest Framework para desenvolvimento de uma API. Atualmente tenho estudado JavaScript e Node.JS.

Em relação as minhas soft-skills, sou uma pessoa bem esforçada e interessada no que me proponho, costumo trabalhar bem em equipe e focar no propósito do time. Tenho facilidade em aprender coisas Estou sempre buscando conhecimento para me manter atualizado no mundo de tecnologia.
